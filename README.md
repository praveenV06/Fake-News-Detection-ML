
# Fake News Detection using Python & Machine Learning

This project uses NLP and ML techniques to detect fake news.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- TfidfVectorizer
- Logistic Regression

## How to Run

1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python fake_news_detect.py`
